# 🚗 Car Price Prediction Using Machine Learning

## 📌 Project Overview
This project predicts the selling price of used cars using Machine Learning techniques. By analyzing factors such as car age, present price, kilometers driven, fuel type, seller type, transmission, and ownership, the model estimates the expected selling price of a car.

## 🎯 Objective
The objective of this project is to build a regression model that accurately predicts used car prices based on historical data.

## 📂 Dataset Features

## ⚙️ Machine Learning Workflow
- Data Collection
- Data Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Data Encoding
- Train-Test Split
- Model Training using Random Forest Regressor
- Model Evaluation
- Price Prediction

## 📈 Model Evaluation
The model was evaluated using:
- Mean Absolute Error (MAE)
- R² Score

The Random Forest Regressor achieved excellent prediction performance on the test dataset.
